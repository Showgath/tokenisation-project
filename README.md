# Rental Income Tokenisation Smart Contract

This repository contains the Solidity smart contract used for my IFTE0007 coursework on asset tokenisation.

## File
- RentalIncomeToken.sol

## Purpose
The contract demonstrates a fungible token model for representing proportional claims on residential rental income.
